Our Enhancement List

  1. Anchor Tag:

    In HTML, an anchor tag is used to create a hyperlink that connects one web page to another or to a specific section within the same page. We implemented this in the 'index.html' page to link to different sections in the 'products.html' page, which saves time for the customers. We used it here (Reference: https://www.w3schools.com/tags/tag_a.asp)

  2. Responsive CSS menu design:

    We have implemented the menu for devices with width smaller than 950px. The menu will turn from horizontal to vertical for those devices. We used @media screen and (max-width:950px){ for this. We used it here.

  3. Responsive CSS index.html "Our products" section design:

    We have implemented the "Our products" section layout for devices with width smaller than 950px. The text will turn from floating right to being in the center and the image will be from floating left to being in the center for those devices. We used @media screen and (max-width:950px){ for this. This will help customer to see the section more clearly. We used it here

  4. Responsive CSS product.html design:

    We have implemented product.html for devices with width smaller than 1024px. The "aside" element will turn from floating right with 25% to cover the page for those devices. The table will be pushed below and cover the whole page. This helps customers to view the information more clearly. We used @media screen and (max-width:1024px){ for this. This will help customers to see the section more clearly. We used it here

  5. Right-sided menu

    Instead of implementing the menu under our logo, we made it in the right side to make it more appealing and clear to customers, using a range of CSS properties. We used it here.
